# Prerequisite
Install `Node.js` and `npm` from https://nodejs.org/en/ 

# Initial Setup
1. Navigate to `/web/client` in terminal
   * run `npm install`
   * run `npm run build`
2. Navigate to `/web` in terminal
   * run `npm install`
   * run `npm run start`

*for client details, refer to `/web/client`*
